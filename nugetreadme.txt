## Digital Results

Thank you for installing Digital Results.  This library adds the ability to return optionally a value or an error without breaking and without adding null checks to your code.

## Setup

There is no special setup required at this time.  Simply reference the base classes in your project.  Documentation has
not yet been released.  However, when it is, we will share a link here.  

## Release Notes
